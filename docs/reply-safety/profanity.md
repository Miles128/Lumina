## 误伤防护

- 「靠」只用感叹/骂人形式：`我靠` / `靠你` / `靠他` / 独立「靠」+ 标点或句界。  
  **不**匹配：挂靠、可靠、依靠、靠谱、靠边、靠着…
- 「傻」只匹配明确脏词形态（傻逼 / 傻B…），**不**匹配：傻子、傻瓜、傻气、装傻…
